# JobPilot AI

A modern, AI-powered job application platform built with React, Vite, and Tailwind CSS.

## Features

- **Dashboard** — Job search overview with stats, activity feed, and quick actions
- **Resume Analysis** — AI-powered resume scoring and suggestions
- **Resume Optimizer** — Content rewriting for ATS compatibility
- **Recruiter Messages** — Inbox for recruiter conversations
- **Interview Coach** — Mock interviews with AI feedback
- **Applications** — Pipeline tracking and management
- **Settings** — Profile, notifications, security, billing, integrations

## Design

- Dark mode by default with light mode toggle
- Primary: Indigo `#6366F1` · Accent: Emerald `#10B981`
- Glassmorphism cards, 16px rounded corners, smooth hover animations
- Fixed left sidebar navigation with top navbar

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19 + Vite 6
- React Router 7
- Tailwind CSS 3
- Lucide React icons
- Plus Jakarta Sans font
